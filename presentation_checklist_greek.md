# Οδηγός Παρουσίασης (Checklist)

Αυτό είναι το "σκονάκι" σου για την ημέρα της παρουσίασης. Ακολούθησέ το για να είσαι σίγουρος ότι όλα θα πάνε ρολόι!

---

## 1. Προετοιμασία (1 ώρα πριν)

Πριν ξεκινήσεις, βεβαιώσου ότι ο Server είναι ξύπνιος.
1.  Άνοιξε το site του σχολείου (ή το `https://104.155.53.205.nip.io`).
2.  Μίλα στο Chatbot: *"Γεια σου"* -> Πρέπει να απαντήσει.
3.  **Test Διαγωνίσματος:** Ζήτα ένα διαγώνισμα ("Δημιούργησε ένα διαγώνισμα") και κατέβασε το PDF.
    *   *Αν δουλεύει:* Είσαι έτοιμος!
    *   *Αν ΔΕΝ δουλεύει:* Μην πανικοβάλλεσαι. Άνοιξε τερματικό και κάνε restart τον server (βλ. Τελευταία ενότητα).

---

## 2. Τι ανοίγουμε στον Υπολογιστή μας (The Setup)

Για να δείξεις "Πώς το έφτιαξα" (Implementation), έχε ανοιχτά τα εξής παράθυρα:

### A. VS Code (Ο Κώδικας)
Έχε ανοιχτά τα "καμάρια" του project:
1.  **`actions/actions.py`:** Για να δείξεις τη λογική (Python). Σκρολάρισε εκεί που λέει `extract_questions_from_docx` (ο Parser) και `ActionCreateExamNew` (ο PDF Creator).
2.  **`domain.yml`:** Για να δείξεις πώς ορίζουμε τις απαντήσεις του Bot.
3.  **`docker-compose.yml`:** Για να δείξεις την αρχιτεκτονική (Rasa + Nginx).

### B. Docker Desktop (Η Υποδομή)
Άνοιξέ το στην καρτέλα **Containers**.
*   *Γιατί:* Δείχνει ωραίο οπτικά να βλέπουν οι κριτές τα 3 κουτάκια (`rasa`, `action-server`, `nginx`) να τρέχουν "πράσινα". Δείχνει ότι είναι ένα σύνθετο σύστημα και όχι ένα απλό script.

### Γ. Browser (Το Αποτέλεσμα)
Έχε ανοιχτές 2 καρτέλες (Tabs):
1.  **Tab 1:** Το Site του Σχολείου (με το Chatbot ανοιχτό κάτω δεξιά).
2.  **Tab 2:** Το Google Cloud Console (προαιρετικά, αν θες να δείξεις ότι τρέχει σε αληθινό server).

---

## 3. Η Ροή της Παρουσίασης (Demo)

Αν σου ζητήσουν Live Demo, κάνε το εξής σενάριο που ξέρουμε ότι δουλεύει άψογα:

1.  **Ο Επισκέπτης:**
    *   Άνοιξε το Chatbot (χωρίς Login στο WordPress).
    *   Γράψε: *"Ποιο είναι το τηλέφωνο;"*.
    *   Γράψε: *"Θέλω διαγώνισμα"*.
    *   *To Bot θα πει:* "Πρέπει να συνδεθείτε". (Δείχνεις την ασφάλεια).

2.  **Ο Μαθητής:**
    *   Κάνε Login στο WordPress (ως Μαθητής/Καθηγητής).
    *   Γράψε: *"Γεια σου"* (Θα σου πει "Γεια σου [Όνομα]!" -> Δείχνεις την αναγνώριση).
    *   Γράψε: *"Θέλω διαγώνισμα Μαθηματικών Α Γυμνασίου"*.
    *   Πάτα το Link, άνοιξε το PDF. (Το "Wow" factor).

3.  **Το "Κουκούλωμα":**
    *   Γύρνα στο VS Code και δείξε το `actions.py`: *"Όλη αυτή η μαγεία γίνεται εδώ, όπου ο αλγόριθμος διαβάζει το Word και φτιάχνει το PDF..."*.

---

## 4. Emergency Kit (Αν κάτι πάει στραβά)

Σπάνιο, αλλά αν την ώρα της παρουσίασης το Chatbot "κολλήσει":

1.  Άνοιξε το τερματικό (Command Prompt/PowerShell).
2.  Συνδέσου με SSH:
    `gcloud compute ssh --zone "us-central1-a" "rasa-server" --project "rasa-school-chatbot"`
    *(Ή όπως αλλιώς μπαίνεις συνήθως)*.
3.  Δώσε την εντολή "Ανάστασης":
    ```bash
    cd rasa-school-chatbot-v2
    sudo docker-compose restart
    ```
4.  Πες στους κριτές: *"Μισό λεπτό να κάνω ένα restart τον Cloud Server"* (Δείχνει άνεση και γνώση!).
