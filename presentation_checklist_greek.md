# Οδηγός Παρουσίασης (Checklist)

Αυτό είναι το "σκονάκι" σου για την ημέρα της παρουσίασης. Ακολούθησέ το για να είσαι σίγουρος ότι όλα θα πάνε ρολόι!

---

## 1. Προετοιμασία (1 ώρα πριν)

Πριν ξεκινήσεις, βεβαιώσου ότι ο Server είναι ξύπνιος.
1.  Άνοιξε το site του σχολείου.
2.  **Check 1 (Greeting):** Γράψε *"Γεια"* -> Πρέπει να βγάλει τα κουμπιά "Μαθητής/Γονέας/Εκπαιδευτικός".
3.  **Check 2 (Search):** Γράψε *"Ψάχνω για εκδρομή"* -> Πρέπει να φέρει Links από το site.
4.  **Check 3 (Study):** Γράψε *"Διάβασμα"* -> Πρέπει να δώσει Links (Public).
5.  **Check 4 (PDF):** Ζήτα ένα διαγώνισμα και κατέβασε το PDF.

---

## 2. Τι ανοίγουμε στον Υπολογιστή μας (The Setup) (Κοινόχρηστη Οθόνη)

Για να δείξεις "Πώς το έφτιαξα", έχε ανοιχτά τα εξής:

### A. VS Code (Ο Κώδικας)
Έχε ανοιχτά τα "καμάρια" του project:
1.  **`actions/actions.py`:**
    *   *Search:* Δείξε την `ActionSearchArticles` (σύνδεση με WordPress API).
    *   *PDF:* Δείξε την `ActionCreateExamNew` (δημιουργία PDF).
2.  **`data/stories.yml`:** Δείξε πώς διαχειριζόμαστε τα Fallbacks (τα σενάρια αποτυχίας).
3.  **`docker-compose.yml`:** Η αρχιτεκτονική (Rasa + Nginx).

### B. Browser (To Live Demo)
Έχε ανοιχτή την ιστοσελίδα του σχολείου.

---

## 3. Η Ροή της Παρουσίασης (Live Demo Script)

Αυτό είναι το σενάριο που καλύπτει ΟΛΕΣ τις νέες λειτουργίες:

### Σκηνή 1: Η Υποδοχή (Role-Based Greeting)
*   **Εσύ:** *"Ας δούμε πώς ξεκινάει η συζήτηση..."*
*   Γράψε: **"Γεια"** (ή πάτα Restart).
*   Το Bot ρωτάει: **"Είστε μαθητής, γονέας ή εκπαιδευτικός;"**.
*   Πάτα: **"Μαθητής"**.
*   *Το Bot απαντά:* "Μπορείς να αναζητήσεις υλικό...".

### Σκηνή 2: Η Αναζήτηση (Search Integration)
*   **Εσύ:** *"Το Bot δεν είναι στατικό. Συνδέεται με το site..."*
*   Γράψε: **"Ψάχνω ανακοινώσεις για εκδρομές"**.
*   *Το Bot:* Ψάχνει στο Google/WordPress και φέρνει αποτελέσματα (Links).
*   **Εσύ:** Κάνε κλικ σε ένα Link για να δείξεις ότι δουλεύει.

### Σκηνή 3: Υλικό Μελέτης (Η Διόρθωση!)
*   **Εσύ:** *"Και για γρήγορο διάβασμα..."*
*   Γράψε: **"Διάβασμα"**.
*   *Το Bot:* "Για ποιο μάθημα;" -> Γράψε "Μαθηματικά".
*   *Το Bot:* Σου δίνει Link. (Εδώ τονίζεις ότι είναι Public!).

### Σκηνή 4: Η Παραγωγή Διαγωνίσματος (Exam)
*   **Εσύ:** *"Και τώρα το κυρίως πιάτο..."*
*   Γράψε: **"Θέλω ένα τεστ Φυσικής Γ Γυμνασίου μέτριο"**. (Δείξε ότι καταλαβαίνει πολλές παραμέτρους μαζί).
*   Το Bot ρωτάει: *"Θέλετε απαντήσεις;"*
*   Γράψε: **"Ναι"** (Για να δείξεις τα Inline Answers).
*   Κατέβασε το PDF και δείξε τις **Μπλε Απαντήσεις** κάτω από τις ερωτήσεις.

### Σκηνή 4: Ανθεκτικότητα (Fallback)
*   **Εσύ:** *"Τι γίνεται αν γράψω αλαμπουρνέζικα;"*
*   Γράψε: **"asdfg hkjh"**.
*   *Bot:* "Συγνώμη, επαναλάβετε;" (1η φορά).
*   Γράψε: **"blablabla"**.
*   *Bot:* "Συγνώμη..." (2η φορά).
*   Γράψε: **"trexa gyreyw"**.
*   *Bot:* "Συγνώμη αλλά δεν κατάλαβα. **Θέλετε να με ρωτήσετε κάτι άλλο;**" (Reset). (Δείχνεις ότι δεν κολλάει).

---

## 4. Emergency Kit (Αν κάτι πάει στραβά)

Σπάνιο, αλλά αν την ώρα της παρουσίασης το Chatbot "κολλήσει":

1.  Άνοιξε το τερματικό.
2.  SSH στον Server.
3.  Δώσε την εντολή "Ανάστασης" (Restart):
    ```bash
    cd rasa-school-chatbot-v2
    sudo docker-compose restart
    ```
4.  Πες στους κριτές: *"Μισό λεπτό να κάνω ένα restart τον Cloud Server"* (Δείχνει άνεση!).
