# Οδηγός Χρήσης Admin App - Διαχείρισης Τράπεζας Θεμάτων

Η εφαρμογή **Admin App** είναι το κεντρικό εργαλείο διαχείρισης της βάσης δεδομένων του Chatbot. Επιτρέπει στους καθηγητές και τους διαχειριστές να επιβλέπουν, να φιλτράρουν και να αναζητούν τις ερωτήσεις που έχουν ανέβει στο σύστημα.

---

## 1. Πρόσβαση στην Εφαρμογή 🔑

### Τοπικά (Στον υπολογιστή σας)
Αν τρέχετε το Chatbot στον δικό σας υπολογιστή:
1.  Ανοίξτε τερματικό.
2.  Τρέξτε: `streamlit run admin_app.py`
3.  Η εφαρμογή θα ανοίξει αυτόματα στο: `http://localhost:8501`

### Απομακρυσμένα (Στον Server/Cloud)
Δείτε τον Οδηγό Google Cloud για λεπτομέρειες σύνδεσης (SSH Tunnel ή Firewall). Συνήθως, η πρόσβαση γίνεται μέσω:
*   **Browser:** `http://104.155.53.205:8501` (αν έχετε ανοίξει Firewall)
*   **SSH Tunnel:** `http://localhost:8501` (αν έχετε κάνει σύνδεση SSH)

---

## 2. Περιήγηση Ερωτήσεων 📂

Μόλις ανοίξετε την εφαρμογή, θα δείτε τον πίνακα **"Περιήγηση Ερωτήσεων"**.
Αυτός ο πίνακας φορτώνει ΟΛΕΣ τις ερωτήσεις που υπάρχουν στη βάση δεδομένων (`questions.db`).

### Τι βλέπετε:
*   **ID:** Ο μοναδικός αριθμός ταυτότητας της ερώτησης.
*   **Subject (Μάθημα):** π.χ. Φυσική, Μαθηματικά.
*   **Class (Τάξη):** π.χ. Β Γυμνασίου.
*   **Difficulty (Δυσκολία):** π.χ. εύκολο, μέτριο, δύσκολο.
*   **Question Text:** Το κείμενο της ερώτησης.
*   **Image Path:** Αν η ερώτηση έχει εικόνα, φαίνεται εδώ το μονοπάτι της.

---

## 3. Φίλτρα & Αναζήτηση 🔍

Στην αριστερή στήλη (Sidebar), υπάρχουν εργαλεία για να βρείτε γρήγορα αυτό που ψάχνετε.

### Φίλτρα (Drop-down Menu)
Μπορείτε να επιλέξετε συγκεκριμένη κατηγορία. Ο πίνακας θα ανανεωθεί αυτόματα.
*   **Μάθημα:** Δείχνει μόνο ερωτήσεις συγκεκριμένου μαθήματος.
*   **Τάξη:** Δείχνει μόνο ερωτήσεις συγκεκριμένης τάξης.
*   **Δυσκολία:** Δείχνει μόνο ερωτήσεις συγκεκριμένου επιπέδου.

> **Tip:** Συνδυάστε φίλτρα! Π.χ. Επιλέξτε "Φυσική" ΚΑΙ "Γ Γυμνασίου" ΚΑΙ "δύσκολο" για να δείτε μόνο τα δύσκολα θέματα Φυσικής Γ' Γυμνασίου.

### Αναζήτηση Κειμένου (Search Box)
*   Γράψτε μια λέξη ή φράση στο πεδίο **"Αναζήτηση Κειμένου"**.
*   Το σύστημα θα ψάξει ΜΕΣΑ στο κείμενο όλων των ερωτήσεων και θα εμφανίσει όσες περιέχουν τη λέξη αυτή.

---

## 4. Προσθήκη Νέας Ερώτησης (Χειροκίνητα) 📝

Αν θέλετε να προσθέσετε μια ερώτηση χωρίς να ανεβάσετε αρχείο Word:

1.  Συμπληρώστε τα πεδία στην ενότητα **"Νέα Ερώτηση"**:
    *   **Μάθημα, Τάξη, Δυσκολία.**
    *   **Κείμενο Ερώτησης:** Γράψτε την εκφώνηση.
    *   **Απάντηση / Λύση:** Γράψτε τη σωστή απάντηση.
2.  (Προαιρετικά) **Εικόνα:** Πατήστε "Browse files" για να ανεβάσετε μια εικόνα/σχήμα που συνοδεύει την ερώτηση.
3.  Δείτε την **Προεπισκόπηση** στα δεξιά για να βεβαιωθείτε ότι όλα είναι σωστά.
4.  Πατήστε το κουμπί **"💾 Αποθήκευση στη Βάση"**.
    *   Θα δείτε μήνυμα επιτυχίας και μπαλόνια! 🎈

---

## 5. Τεχνικές Πληροφορίες 🛠️

*   **Βάση Δεδομένων:** Η εφαρμογή διαβάζει απευθείας το αρχείο `data/questions.db`.
*   **Refresh:** Αν ανεβάσετε νέο υλικό μέσω του Chatbot, κάντε ένα **Refresh (F5)** στον browser για να δείτε τις νέες ερωτήσεις στον πίνακα.

---

## 6. Διαγραφή Ερωτήσεων 🗑️

Στο κάτω μέρος της εφαρμογής υπάρχει η ενότητα **"Διαγραφή Ερώτησης"**:
1.  Πατήστε το βελάκι για να ανοίξει η ενότητα (Expansion).
2.  Επιλέξτε από τη λίστα την ερώτηση που θέλετε να σβήσετε (εμφανίζεται ID και κείμενο).
3.  Πατήστε το κόκκινο κουμπί **"Οριστική Διαγραφή"**.
    *   Η σελίδα θα ανανεωθεί και η ερώτηση θα εξαφανιστεί από τον πίνακα.
