# Οδηγός Docker Desktop

Αυτό το έγγραφο εξηγεί τον ρόλο του Docker Desktop στον υπολογιστή σας (Windows) και πώς το χρησιμοποιούμε για να εξομοιώσουμε το περιβάλλον του Server.

---

## 1. Τι είναι το Docker Desktop;

Φανταστείτε το Docker σαν ένα **φορτηγό πλοίο** και την εφαρμογή μας σαν ένα **κοντέινερ (container)**.
*   Μέσα στο κοντέινερ υπάρχει όλο το λογισμικό (Python, Rasa, βιβλιοθήκες) που χρειάζεται το Chatbot για να δουλέψει.
*   Το **Docker Desktop** είναι το λιμάνι στον υπολογιστή σας που "τρέχει" αυτά τα κοντέινερ.

**Γιατί το χρησιμοποιούμε;**
Εξασφαλίζει ότι το Chatbot θα τρέχει **ακριβώς το ίδιο** στον υπολογιστή σας (Windows) και στον Server (Linux). Αν δουλεύει στο Docker, δουλεύει παντού ("It works on my machine" problem solved).

---

## 2. Βασικές Ρυθμίσεις (Settings)

Μια κρίσιμη ρύθμιση για το Rasa (που είναι "βαρύ" λογισμικό) είναι η μνήμη RAM.

**Πού θα το βρείτε:**
1.  Ανοίξτε το Docker Desktop.
2.  Κάντε κλικ στο γρανάζι (Settings) πάνω δεξιά.
3.  Πηγαίνετε στο **Resources**.
4.  **Memory Limit:** Βεβαιωθείτε ότι είναι τουλάχιστον **4 GB** (ιδανικά 6 GB). Αν είναι λιγότερο, το Rasa μπορεί να "κολλήσει" κατά την εκπαίδευση.
5.  **Swap:** Αυξήστε το στο 1 GB (βοηθητική μνήμη).

---

## 3. Διαχείριση Αρχείων (Volumes)

Το Docker είναι "κλειστό κουτί". Τα αρχεία μέσα του χάνονται όταν το κλείσουμε, **ΕΚΤΟΣ** αν χρησιμοποιήσουμε Volumes.

*   Στο αρχείο `docker-compose.yml`, ορίσαμε ότι ο φάκελος του υπολογιστή σας (`./data`) θα "καθρεφτίζεται" μέσα στο Docker (`/app/data`).
*   Αυτό σημαίνει ότι όταν αλλάζετε ένα αρχείο στον φάκελο του Project, το Docker το "βλέπει" αυτόματα (αν κάνετε restart).

---

## 4. Χρήσιμες Εντολές & Troubleshooting

Αν τα πράγματα "κολλήσουν", εδώ είναι τα εργαλεία σας (μέσα από το τερματικό):

*   `docker ps`: Δείχνει ποια κοντέινερ τρέχουν τώρα (σαν τη Διαχείριση Εργασιών).
*   `docker-compose up`: Ξεκινάει την εφαρμογή.
*   `docker-compose down`: Σταματάει και σβήνει την εφαρμογή.
*   `docker system prune`: **ΠΡΟΣΟΧΗ!** Σβήνει όλα τα παλιά/σταματημένα κοντέινερ και ελευθερώνει χώρο στον δίσκο σας. Χρήσιμο αν γεμίσει ο δίσκος.

---

## 5. Το Περιβάλλον Containers

Στην καρτέλα **Containers** του Docker Desktop, μπορείτε να δείτε την ομάδα `rasa-school-chatbot-v2`. Αν πατήσετε πάνω της, θα δείτε:
1.  `rasa`: Ο εγκέφαλος.
2.  `action-server`: Τα χέρια (Python κώδικας).
3.  `nginx`: Η πόρτα (Web Server).

Αν κάποιο έχει κόκκινο τετράγωνο, σημαίνει ότι σταμάτησε (crash). Πατώντας πάνω του, βλέπετε τα **Logs** (καταγραφή σφαλμάτων) για να καταλάβετε τι πήγε στραβά.
